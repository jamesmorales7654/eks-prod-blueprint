# EKS Production Blueprint
Production-ready EKS cluster with Terraform — modular VPC, IAM, and remote state.

This project demonstrates a secure baseline for running Kubernetes workloads in AWS using infrastructure-as-code.

## Features

- Modular Terraform architecture
- Secure VPC networking
- IAM least-privilege roles
- Remote Terraform state
- Logging and monitoring integration
- Security-focused cluster configuration

## Purpose

This repository is a reference architecture for deploying a secure Kubernetes environment in AWS.

It reflects patterns used in production cloud environments where security, repeatability, and automation are critical.

## Technologies

- AWS
- Terraform
- Kubernetes (EKS)
- IAM
- VPC
